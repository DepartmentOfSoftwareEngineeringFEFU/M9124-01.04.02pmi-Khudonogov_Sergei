# Ice Navigator — Система поддержки принятия решений для ледовой навигации

Веб-приложение для визуализации ледовой обстановки в Арктике и построения оптимальных маршрутов судов с учётом концентрации и толщины льда.

## Основные возможности

- Автоматическая загрузка ежедневных данных концентрации морского льда (AMSR2, Университет Бремена)
- Загрузка и обработка данных о возрасте льда (NSIDC Ice Age v4), преобразование в толщину
- Интерактивная карта с растровым и точечным слоями концентрации и толщины льда
- Построение оптимального маршрута с помощью алгоритма A* с учётом ледового класса судна
- Управление справочниками портов, судов и сохранённых маршрутов
- Визуализация, скрытие/отображение и удаление маршрутов на карте, изменение цвета маршрутов
- Поддержка бесконечного горизонтального скроллинга карты
