# Разработка программной системы поддержки принятия решений судоводителя для планирования движения судов в ледовых условиях с использованием больших ретроспективных данных

Веб-приложение для визуализации ледовой обстановки в Арктике и построения оптимальных маршрутов судов с учётом концентрации и толщины льда.

## Основные возможности

- Автоматическая загрузка ежедневных данных концентрации морского льда.
- Обработка данных о возрасте льда (NSIDC Ice Age v4), преобразование в толщину.
- Интерактивная карта с растровым и точечным слоями концентрации и толщины льда.
- Построение оптимального маршрута с помощью алгоритма A* с учётом ледового класса судна.
- Управление справочниками портов, судов и сохранённых маршрутов.
- Визуализация, скрытие/отображение и удаление маршрутов на карте.

## Установка
Для установки используется Docker.
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/D-Bawww/web_ice.git
2. Перейдите в папку с проектом.
3. Запустите приложение:
   ```bash
   docker compose up --build
4. Откройте в браузере:
   ```bash
   http://localhost:5000
